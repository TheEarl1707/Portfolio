EKD Cinema (Sep 2022 - Oct 2022)
Description:

A PHP-based project that aims to simulate an online ticket purchasing website wherein credit-based currency can purchase additional add-ons along with the movie.

Features and Functions:

If the user is logged in and makes a transaction, the amount will be deducted from their virtual wallet, and they will earn reward points relative to how much they spent from the tickets

If a transaction is made without an account, then it will prompt the user to enter an amount. This amount should not be less than the total price of the transaction, or it will not proceed. The system will also send a receipt of the transaction to the provided email

The administrators of this website can add, update and delete movies and add-ons, as well as an archive option for movies, and a graph that can display the amount of tickets sold of each movie on the specified day/s
